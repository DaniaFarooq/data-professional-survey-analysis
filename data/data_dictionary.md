# Survey Data Dictionary

## Respondent Information
| Column | Type | Description | Example |
|--------|------|-------------|---------|
| Respondent_ID | String | Unique anonymous identifier | RESP_0001 |
| Job_Title | Categorical | Current professional role | Data Analyst |
| Salary_USD | Integer | Annual salary in USD | 75000 |
| Salary_Range | String | Categorical salary bracket | 66k-85k |
| Age | Integer | Respondent age | 32 |
| Gender | Categorical | Self-reported gender | Male |
| Country | Categorical | Country of residence | United States |

| Column | Type | Description |

| Difficulty_Breaking_In | Categorical | Entry difficulty rating | Difficult |

| Programming_Language | Categorical | Primary programming language | Python |
